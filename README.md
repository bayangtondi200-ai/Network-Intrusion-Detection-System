
# Network Intrusion Detection System

IDS léger développé en Python avec Scapy.

## Fonctionnalités

- Détection SYN Flood
- Détection Port Scan
- Journalisation des alertes
- Surveillance en temps réel

## Installation

pip install -r requirements.txt

## Exécution

sudo python3 ids.py
